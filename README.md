## <img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="25" /> &nbsp;프론트엔드 개발자 김진규입니다.

✔️ &nbsp;**사회적 문제 해결에 기여할 수 있는 서비스**에 관심이 많습니다.\
